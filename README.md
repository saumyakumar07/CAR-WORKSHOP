# 🚗 CarWorks — Car Workshop Management System

A fully functional **car workshop management web app** built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies, no server required — just open `index.html` in any browser.

---

## 📸 Preview

> Book services, generate bills, and track customer records — all in one page.

---

## ✨ Features

- **New Booking Flow** — Collect customer details (car number, name, address, contact, zip)
- **5 Service Categories** with dynamic pricing based on car make & fuel type
- **Auto Bill Generation** with 10% discount on bills above ₹10,000
- **Customer Records** — All bookings saved locally, searchable by car number or name
- **Print Bill** — One-click print-ready bill
- **Offline-ready** — No internet connection needed after loading

## 📌 Origin

This project was originally a **C++ console application** that used flat `.txt` files for data storage and ran in a Windows terminal. It has been fully converted to a modern web app while preserving all original business logic and pricing.
